# Pokémon Python Game

Pequeno jogo inspirado em Pokémon desenvolvido em Python durante meus estudos na plataforma Solyd.

## Funcionalidades

- Exploração do mapa
- Batalha contra inimigos
- Sistema de captura de Pokémon
- Sistema de salvar e carregar jogo

## Tecnologias

- Python
- Programação Orientada a Objetos
